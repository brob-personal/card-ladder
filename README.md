## Testing

Run all tests from the project root with Python's built-in `unittest` discovery:

```bash
python -m unittest discover -s tests
```
## Prototype 1 Features

- Playable Hearts game vs Alfred (AI opponent)
- Trait system (Thief, Medium, Stoic, Duelist)
- Table modifiers (Fog on the Moor, Marked Deck)
- Alfred dialogue system with event triggers
- Terminal-based UI
- Unit test suite for core logic
